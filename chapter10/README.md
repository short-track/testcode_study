# 챕터 10

챕터 10을 읽고 실전 연습을 위한 문제입니다. 챕터 10은 e2e 테스트를 하기 위해 어떤 환경을 갖춰서 진행하는 것이 필요한지 알려주고 있습니다.
특히 테스트를 위한 서버환경을 Docker를 활용하여, 매번 깨끗한 서버환경에서 테스트를 할 수 있도록 합니다.

아래와 같이 진행해봅시다.

1. 먼저! 해당 프로젝트의 페이지를 띄워보시요. (실행시키기)
2. 현재 각 POST 페이지에는 댓글 기능이 있습니다. 이 댓글을 실제로 생성하려고 하면 오류가 발생합니다. IDE 에서 `코드를 작성하시오` 주석을 검색해서, 해당 주석이 처리된 곳들에 코드를 채워넣으세요.
3. 이젠 댓글 기능과 관련된 e2e 테스트를 작성해봅시다. IDE 에서 `테스트를 작성하시오` 주석을 검색해서 테스트 코드를 작성하고 모두 통과시켜주세요.
